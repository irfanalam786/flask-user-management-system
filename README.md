
# 🚀 User Management Web Application (Flask Full Stack Project)

## 📌 Project Overview
This project is a beginner-friendly **Python Full Stack Web Application** built using **Flask, HTML, CSS, and SQLite**.

The application allows users to:
- Add user details
- View all users in a table
- Store data permanently in SQLite database

This project demonstrates how **frontend, backend, and database work together** in real-world web applications.

---

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS
- Database: SQLite
- Tools: VS Code, Git, GitHub

---

## ⭐ Features
✅ Add new user  
✅ Display users in table  
✅ Persistent SQLite storage  
✅ Clean UI  
✅ Auto database + table creation
